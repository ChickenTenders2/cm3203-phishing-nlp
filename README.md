# CM3203 Phishing NLP Project

Comparative evaluation of NLP approaches for phishing email detection.

## Datasets
- MeAJOR
- Human-LLM generated phishing-legitimate emails

## Current stage
- Dataset inspection and preprocessing design

## Folder structure
- data/raw/: original downloaded files
- data/processed/: cleaned and split outputs
- notebooks/: experimental notebooks
- src/: reusable code
- results/: saved metrics and outputs
- models/: trained model artifacts
